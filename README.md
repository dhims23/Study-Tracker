# Study-Tracker 📚

A web-based application independently developed to help students efficiently log, monitor, and optimize their daily study sessions. 

# Overview
As a student transitioning into higher education, I noticed that maintaining consistency and tracking learning retention is a universal challenge. This project was born out of a desire to create a functional tool that acts as a stepping stone toward my ultimate vision: building an AI-driven "Second Memory" system to mitigate information loss in human learning.

# 🛠️ Tech Stack
* **Backend Logic:** Python 
* **Frontend Interface:** HTML5 & CSS3 
* **Development Environment:** Visual Studio Code (VS Code) 

# ✨ Key Features
* **Session Logging:** Seamlessly input and record daily study durations and topics.
* **Data Handling:** Basic data structures implemented via Python to process user inputs efficiently.
* **User-Centered Design:** A clean, intuitive, and distraction-free interface built from scratch using custom CSS.

#  Future Roadmap
This repository represents my foundational step in software development. Moving forward, I plan to:
1. Integrate a SQLite database to handle long-term data persistence.
2. Implement basic predictive analytics using Python to suggest optimal study times based on historical data.
